TuringMachine module¶

TuringMachine.Begin(commands: list)¶

Main method

Calls TerminalPainter.DrawTerminal() and ExecuteCommand() in order to execute given commands and draw the output to the terminal. Also handles delays between commands if step-by-step mode is disabled

Parameters

commands (list) – list of commands to execute

TuringMachine.Entry()¶

Program entry point

Gets parsed arguments from methods ParseArguments() and Compiler.CommandLists() respectively and passes them to Begin()
